Cab Booking Application

Development

Designed a GUI application for booking a cab with different interfaces for both driver and customer. Both entities are provided with a wallet to add and transact money.
The nearest cab was assigned to the passenger (using Djiktras algorithm while assuming the roads formed a graph with the pickup point as nodes) and a fare with an estimated time of arrival was calculated for the passenger.
The project was built using Java, PHP and MySQL
